== GET /purchaseLists ==

The purchaseLists Endpoint will get all purchase List properties back. From here you can access all the purchase items
from a purchase list.

[cols="1,1,2", options="header"]
.Purchase list properties
|===
|Name
|Datatype
|Description

|name
|text
|Title of the purchase list

|date
|https://en.wikipedia.org/wiki/ISO_8601[ISO 8601]
|Date of shopping
|===
